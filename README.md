# Contributor Handbook: Contribution Pathways

This repository manages the Pathways project in the [WordPress Contributor Handbook](https://make.wordpress.org/handbook/). This is a collection of concise guides and (soon) Good First Issues that help new contributors find their way into the WordPress project.

## Status
- **Help with Testing:** [Walk through one of these guides](https://github.com/orgs/WordPress/projects/282/views/6) and leave feedback
- **We need guides:** If you can, please pick up an idea from the [project board](https://github.com/orgs/WordPress/projects/282/views/1) and flesh it out [using these steps](https://make.wordpress.org/handbook/pathways/write/create-a-pathway-guide/).
- **Help us Improve:** We're working with WordPress Markdown Importer, but would love ideas on how we can better organize and surface pathway guides, and make them look nicer too. Come discuss it with us in [#core-program](https://wordpress.slack.com/archives/core-program)!
- **Suggest a new pathway:** To suggest a new pathway, [open an issue](https://github.com/WordPress/contributor-handbook/issues/new/choose). Then feel free to [draft the guide](https://make.wordpress.org/handbook/pathways/write/create-a-pathway-guide/) if you can.

### A note on scope

Pathway guides are maps, not manuals. They're meant to give contributors enough detail to find what they need and get started, but they don't rewrite or supercede existing handbook pages. When in doubt, link out.

## Questions?
