# All pathway guides

<!-- wp:block {"ref":6191} /-->

Each guide walks you through a specific way to contribute, with steps and links to get you going.  You can also check the [Good First Issues board](https://github.com/orgs/WordPress/projects/289/views/1?pane=info) for tasks you can pick up right now.

New to contributing? [Get set up before you dive in](https://make.wordpress.org/handbook/pathways/before-you-begin/): you'll get accounts and best practices, plus how to find help and earn contributor badges.

Teams that want to add their own guides can [learn how here](https://make.wordpress.org/handbook/pathways/about/).
