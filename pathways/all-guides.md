# All pathway guides

Each guide walks you through a specific way to contribute, with steps and links to get you going.  You can also check the [Good First Issues board](https://github.com/orgs/WordPress/projects/289/views/1?pane=info) for tasks you can pick up right now.  
  
<p><strong>New to contributing?</strong> <a href="https://make.wordpress.org/handbook/pathways/before-you-begin/">Get set up before you dive in.</a> You'll get accounts and best practices, plus how to find help and earn contributor badges.</p>

<p>Teams that want to add their own guides can <a href="https://make.wordpress.org/handbook/pathways/about/">learn how here</a>.</p>
