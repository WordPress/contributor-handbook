# Bulk Enrollment of Students

If the institution wants to provide student data for **batch import and enrollment**, they must submit a table containing at least the following information for each student:

* **Full name (first name and surname)**  
* **Email address**  
* **Languages spoken by the student:**  
  * If only one language: enter the language in English.  
  * If more than one language: enter the languages in English, separated by commas and enclosed in double quotation marks (e.g., “Spanish, English, Portuguese”).  
* **Preferred language:** Apply the same rules as for *Languages spoken by the student*.  
* **Preferred start date:** Use the format **DD/MM/YYYY**.  
* **End date:** Use the format **DD/MM/YYYY**.  
* **Accessibility needs:** Leave blank if there are none.
